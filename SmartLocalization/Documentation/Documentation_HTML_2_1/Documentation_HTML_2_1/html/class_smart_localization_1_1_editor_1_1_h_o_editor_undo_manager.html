Detailed Description

Editor undo manager. To use it:

  1. Store an instance in the related Editor Class (instantiate it inside the

OnEnable method). Call undoManagerInstance.CheckUndo() BEFORE the first UnityGUI call in OnInspectorGUI. Call undoManagerInstance.CheckDirty() AFTER the last UnityGUI call in OnInspectorGUI.
